# 💰 AI Financial Assistant

An AI-powered system that transforms raw bank statements into meaningful financial insights and helps users make smarter financial decisions.

---

## 🚀 Problem

Bank statements are difficult to understand and provide little actionable insight.

- Users don’t know where their money goes  
- Manual expense tracking is tedious  
- Financial decisions are often made blindly  

👉 Result: Poor financial awareness and habits  

---

## 💡 Solution

This project provides a **one-click financial analysis system**:

- Upload your bank statement (CSV / PDF)
- Automatically extract and analyze transactions
- Get insights, financial score, and advice
- Simulate better financial decisions

👉 No manual tracking required

---

## ⚙️ How It Works

1. Upload bank statement  
2. Parse transactions (CSV / PDF)  
3. Categorize transactions using **AI + rule-based logic**  
4. Calculate financial metrics  
5. Generate financial health score  
6. Provide AI-based advice  
7. Simulate better spending decisions

## 🏗️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- PDF Parsing (pdfplumber)

---

## ⚠️ Limitations

- Currently optimized for HDFC bank format  
- Some UPI transactions may be categorized as "Other"  
- Cash spending requires manual input for full accuracy
